#### 思路：

- 正则判断输入正确与否
- 传入字符串分割进行判断是否正确
- 正确就保存
- 取消的话也是分割判断



>README ⽂档中应描述如何运⾏单元测试或主程序来证明作业的正确性（⾄少针对测试⽤例输⼊能够
得到对应输出）

格式为 {⽤户ID} {预订⽇期 yyyy-MM-dd} {预订时间段 HH:mm~HH:mm} {场地} ，

场地A的时间段
